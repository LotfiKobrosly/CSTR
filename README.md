# Continuous Monte Carlo Tree Search
This repo contains implementations of continuous versions of Nested Monte Carlo Tree Search and Nested Rollout Policy Adaptation, which are tested within the [`pc-gym`](https://maximilianb2.github.io/pc-gym/) library, which presents a few chemistry environment models. The goal is to apply the algorithms and compare their performances, potentially with a baseline.
